# Installation

- Drag and drop into your resources folder
- Do 'enable truck-job'
- DELETE the -main part of the folder

# version 

Version - 1.0.0

# Features

1. Typing the command 'trucking) to get started
2. Vehicle automatically spawns
3. Trailer spawns automatically
